#KayoohTerus

Filosofi & Nilai Jual

Kayooh adalah jawaban atas kebutuhan akan privasi dan kedaulatan data di era digital. Sebagai aplikasi pelacak gowes mandiri, jejak rute dan statistik Anda tidak akan pernah dikirim ke server pihak ketiga untuk diolah atau dijual. Semua data tersimpan aman di server pribadi Anda.

Analisis Performa

✅ Kelebihan

  • Privasi Mutlak: Kontrol penuh atas data aktivitas pribadi.
  • Ringan & Cepat: Antarmuka bersih tanpa beban fitur sosial yang berat.
  • Bunker Mode: Tetap berjalan stabil meskipun melewati area tanpa sinyal internet.
  • Multi-Flexing: Ekspor statistik gowes dengan desain transparan yang profesional.

⚠️ Kekurangan

  • Screen-On Required: Layar HP harus tetap menyala selama perekaman agar GPS tidak ditidurkan oleh sistem Android.
  • Konsumsi Baterai: Layar yang menyala terus-menerus berdampak pada penggunaan daya baterai yang lebih tinggi.
  • Wajib HTTPS: Sensor GPS hanya akan berfungsi jika domain menggunakan SSL yang valid.

Fitur Unggulan

📡 Live GPS Tracker PRO

Rekam rute langsung dari browser HP dengan dukungan Smart Auto-Pause yang akan menjeda timer secara otomatis saat Anda terhenti di lampu merah.

🔄 Strava Sync

Impor riwayat gowes lama Anda dari Strava ke database mandiri Kayooh hanya dengan beberapa klik.

Panduan Instalasi Server

  1. Unggah source code ke direktori web server Anda.
  2. Atur hak akses direktori agar PHP bisa menulis database: 
    sudo chown -R www-data:www-data /folder-kayooh
sudo chmod -R 755 /folder-kayooh
  3. Akses domain Anda untuk memulai proses Setup Akun Admin.

Panduan Impor dari Strava

  1. Buat aplikasi API di Strava API Settings.
  2. Masukkan Client ID & Secret di halaman Sync Strava pada Kayooh.
  3. Gunakan tombol Dapatkan Token Sakti untuk otorisasi.
  4. Klik Sinkronisasi Sekarang untuk menarik data aktivitas.
